# Urban Routes Testing Project

## 📌 Project Description
This project includes manual testing performed on the Urban Routes web application.  
The objective was to validate map functionality, UI behavior, and core user interactions.

## 🎯 Scope
The following features were tested:

- Map navigation and zoom
- Address input fields ("From" and "To")
- Map objects (parks, landmarks, metro stations)
- Fullscreen behavior
- UI elements
- Application logo behavior

## 🛠 Tools Used
- Jira (bug tracking)
- Excel (test case execution)
- Chrome Browser
- Windows 11

## 📊 Test Execution Summary
- Total test cases: 24
- Bugs found: 9
- Evidence: screenshots and screencasts included

## 📁 Project Files
- Test cases documentation
- Bug evidence (screenshots and screencasts)
- Jira bug screenshots

## 🔗 Traceability
Each failed test case includes a direct link to its corresponding Jira ticket to ensure traceability between test execution and defect reporting.
